# 逃避型依附的形成：去活化策略（Deactivating Strategy）

## 核心定義

逃避型依附是對**拒絕與負向行為**的適應性反應。它反映一種**去活化策略**——關閉依附系統、最小化依附需求與感受，以避免拒絕、侵入與羞辱，藉此保全自我。

> 這些病人「去活化了他們的關係」，選擇看似拋棄連結。我們說「看似」，因為去活化其實是一種**讓依附與連結在他們能力所及範圍內發生**的策略——代價是迴避真實關係帶來的情緒痛苦。（ch07, pp. 15–26）

## 形成機制

### 照顧者特徵：貶抑（Dismissive）的心智狀態
- 貶抑自己與孩子的依附需求，以維持自身狀態的現狀（Main et al., 1985）
- 拒絕性、侵入性、羞辱性的行為特徵

### 嬰兒的適應
- 在家觀察中，逃避型嬰兒其實**哭得比安全型嬰兒多**（與矛盾型相當）——分離苦惱並不少
- 但在陌生情境（Strange Situation）中，他們**不哭**、重逢時**避開母親**
- Main（1995）：「依附行為被主動迴避取代……移開視線、走開、轉身、從懷抱中掙脫，以及持續注意無生命環境」
- Spangler & Grossmann（1993）生理研究：這些嬰兒的「遊戲」**心率不降**——不是真正投入遊戲，而是**把注意力從母親身上移開**

### 防衛性排除（Defensive Exclusion）
- Bowlby（1980）：孩子排除會觸發照顧者焦慶的經驗面向
- 代價：大片人類經驗被排除在意識之外，以維繫與照顧者的連結——卻以成長與發展為代價
- 當防衛機制被長期依賴，結果是**人格的侷限與收縮**

## 偽自我依賴（Pseudo Self-Reliance）

- 關閉覺察 ≠ 解決需求；關閉情緒 ≠ 管理情緒
- 當自我為了避免受傷而轉開人群、為了避免潰堤而關閉情緒、獨自忍受——自我成為一個**封閉系統、不可持續的生態**
- 當恐懼、絕望、憤怒的情緒被關閉，**正向情緒**（喜悅、創造、自我效能、關係連結的愉悅）也一同萎縮

> Kaplan（1987）分離焦慮測驗：逃避型 6 歲孩子被問「分離時能做什麼」時回答「我不知道」「沒辦法」「逃跑」——無法構思任何策略來幫自己因應感受，即使能命名那些感受。

## 成人初現（Initial Presentation）

- 多因**伴侶的絕望與挫折**（伴侶想要更多連結）而被「送」來治療
- 貶抑依附關係的重要性——保護策略在生活中重演
- Fonagy & Target（2007）：貶抑表現為「敘事的貧瘠」「心理世界的空洞」
- 「我不知道」「我不記得」「就……很正常」——是**逃避型嬰兒重逢時轉身離開的身體姿態**在語言中的重現
- 與內在、身體、非語言經驗斷線；難以命名自己想從治療中得到什麼

## 臨床關鍵點（供對話練習註解引用）

| 概念 | 內涵 |
|------|------|
| 去活化策略 | 關閉依附系統以避免拒絕 |
| 防衛性排除 | 排除威脅照顧者的經驗面向 |
| 偽自我依賴 | 轉開人群≠解決需求 |
| 「dealing, but not feeling」 | 能在世上運作，卻與感受斷線 |
| 移開注意力 | 不是轉向玩具，而是**從母親身上移開** |
| 短期 vs 長期交換 | 去活化短期降低被拒絕的痛；長期導致情感與關係生活的萎縮 |
